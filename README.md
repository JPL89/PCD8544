## Atmega8 pin connection

<ul>PB1 --> CS (SCE) Chip Enable<ul/>
<ul>PB2 --> DC (data-command)<ul/>
<ul>PB3 --> SDI (MOSI)<ul/>
<ul>PB5 --> SCK (SCL)<ul/>
